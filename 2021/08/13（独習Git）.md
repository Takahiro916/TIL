## 【独習Git】  

### 【Chapter 4　リポジトリの作り方と使い方】  
**`4.1 リポジトリの基礎を理解する`**  
- リポジトリはそこで行う作業が追跡管理される特別なストレージ領域。  

**`4.2 git initで新しいリポジトリを作る`**  
```
% cd
% mkdir buildtools （作業ディレイクトリ作成）
% cd buildtools
% git init （リポジトリ作成）
% ls
```
